```

│   ...
├── frontend
│   ├── data
│   ├── domain
│   ├── components
│   ├── pages
│   ├── stores
│   ├── ...
│   └── Dockerfile
├── compose.yaml
└── README.md

```

## Run the Frontend, Backend and MongoDB, using the Docker compose
In the root repository, execute the docker compose:
```
docker compose up -d
```
